# runebox

Alias package: installing `runebox` installs
[`runebox-sync`](https://pypi.org/project/runebox-sync/), the standalone sync
client for the [Runebox](https://runebox.ai) registry. Both names give you the
`runebox-sync` command.

See the [runebox-sync package](https://pypi.org/project/runebox-sync/) or
[runebox.ai](https://runebox.ai) for docs.
